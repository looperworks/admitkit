# AdmitKit

A search tool for US graduate admissions data. Query admission requirements, deadlines, and program details for master's and doctoral programs across 6,000+ institutions.

**Live site:** [looperworks.github.io/admitkit](https://looperworks.github.io/admitkit/)

## Features

- Search US institutions by school, state, degree type, and region
- View program-level admission requirements, deadlines, and application details
- Save and compare programs
- Data sourced from the U.S. Department of Education API
