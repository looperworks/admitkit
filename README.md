# AdmitKit

Find your path to graduate school. Search admission requirements, deadlines, and program details for master's and doctoral programs at any US university.

**Live site:** [looperworks.github.io/admitkit](https://looperworks.github.io/admitkit/)

## Features

- Search 6,000+ US institutions for graduate program information
- Filter by school, state, degree type, and region
- View admission requirements, deadlines, and program details
- Save and compare programs
- Dark mode support
- Powered by live data from the U.S. Department of Education API
